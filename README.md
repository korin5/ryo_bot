# ryo_bot

ryo_bot 是基于 [ICQQ](https://github.com/icqqjs/icqq) 开发的QQ机器人

为解决QQ琴友的找谱需求，ryo_bot 诞生了

使用方法请查阅 [用户文档](https://ryobot.netlify.app/use.html)
