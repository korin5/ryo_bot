# ryo_bot

ryo_bot 是基于 [ICQQ](https://github.com/icqqjs/icqq) 开发的QQ机器人

ryo 的名字来源于动漫孤独摇滚中的角色 [山田凉](https://mzh.moegirl.org.cn/山田凉)，为什么要取这个名字？仅仅是因为我喜欢罢了（笑）
